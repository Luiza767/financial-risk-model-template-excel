# Risk Management Financial Model (Excel)

## Overview
This project presents a financial risk management model built in Excel.

The model integrates revenue projections, profitability analysis, and debt structure to assess financial performance and risk over a multi-year horizon.

## Objective
To evaluate financial stability, cash flow generation, and the impact of leverage on overall risk.

## Model Structure

### 1. Assumptions Sheet
Key input parameters:
- Initial investment
- Revenue level and growth rate
- Profit margin
- Discount rate
- Debt level
- Interest rate
- Debt repayment period
- Capital expenditures (CapEx)
- Tax rate
- Working capital
- Long-term growth rate

### 2. Financial Model (Years 1–3)
- Revenue projections
- EBITDA estimation
- Cash flow calculation (CFC)

### 3. Debt Model
- Initial debt balance
- Interest expense
- Debt repayments
- Closing debt balance

## Methodology
The model combines financial forecasting with debt modeling to evaluate risk exposure and financial sustainability.

## Use Cases
- Financial planning and analysis (FP&A)
- Risk assessment
- Corporate finance modeling
- Investment evaluation

## Tools
- Microsoft Excel
